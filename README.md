# Application-Carnet-d-adresses


Cette application est un carnet d’adresses permettant la gestion complète de contacts, connectée à une base de données locale SQLite.
Elle a été développée en Python avec une interface graphique PyQt6.


Fonctionnalités :
Ajout d’un contact (nom, prénom, téléphone, courriel)

Modification d’un contact existant
Suppression d’un contact
Sélection d’un contact directement depuis le tableau
Mise à jour immédiate des données dans la base SQLite
Gestion des erreurs (champs obligatoires, doublons, saisies invalides)
